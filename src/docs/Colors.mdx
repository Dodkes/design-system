import { ColorItem, ColorPalette, Meta } from "@storybook/blocks";

<Meta title="Design System/Colors" />

You can also add a color palette to your documentation. This is a great way to show the colors used in your design system.

In `src/docs/Colors.mdx` file, add the following content:

## Brand colors

<ColorPalette>
    <ColorItem
        title="new-brand"
        subtitle="--variable-name-xx"
        colors={["#000000", "#FFFFFF"]}
    />
</ColorPalette>
